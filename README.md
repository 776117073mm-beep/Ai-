# AI Creative Studio

This repository contains the foundation of a modular AI Creative Platform monorepo.

## Included foundation

- Monorepo workspace structure for apps and shared packages
- Web application shell for the creative studio experience
- UI package with a modern home screen and a professional video workspace placeholder
- Shared AI core package with an orchestrator and engine contracts
- Workflow package for future automation and orchestration
- Architecture documentation for the long-term platform strategy
- Video engine adapter placeholders for FFmpeg, MLT, OpenTimelineIO, and OpenFX

## Getting started

1. Install dependencies: npm install
2. Start the web experience: npm run dev
3. Open the local Vite app in your browser
